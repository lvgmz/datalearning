- aggregate functions - COUNT, SUM, MIN and MAX, AVG
- used typically as - SELECT aggregatorfunction AS easynametovisualize FROM dataset
- column names can be changed visually using an alias - columnname AS nameshown
- aggregators only aggregate vertically
- query clause order - SELECT, FROM, WHERE, GROUP BY, HAVING, ORDER BY

recap

- COUNT() - used to return number of non null rows in a column, * for all rows or column name for specific column
- SUM() - used to return total of all values in a given column, can only be used on columns containing numerical values, nulls treated as 0
- AVG() - calculates average from a selected group of values, can only be used in vertical colulmns, ignores nulls
- MIN()/MAX() - returns lowest/highest value in a column, can be used in non numerical columns with strings alphabetically and dates in order
- GROUP BY - groups rows that have the same value into summary rows, often used with aggregate functions
- HAVING - used to filter an aggragated query

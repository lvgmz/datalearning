### Kaggle 4 - Grouping and Sorting
- groupby() is used to group data to perform specific tasks on the group
	- each group is a slice of the DataFrame containing only data with matching values
	- you can group by multiple columns
	- agg() - groupby method that lets you run different functions on the DataFrame simultaneously
- multilevel index - way of indexing dealing with multiple levels requiring 2 labels
	- reset_index()  - used to convert back to regular index
- grouping returns data in index order not value order, sort_values() helps us get the data in the order that we want
	- sort_index() - is used to sort index, used the same as sort_values
	- can sort by more than one column at a time
### Kaggle 5 - Data Types and Missing Values
- dtype - data type for a column in a DataFrame or a Series
	- can use dtype property to get type for a specific column or on its own to get dtype of every column in a DataFrame
	- strings are given the object type
	- can convert to other sensible types using astype()
- missing data entries are given the value NaN - Not a Number, and its always float64 dtype, methods for missing data
	- isnull() - checks for null
	- fillna() - replace missing values
	- replace() - used to make changes 
### Kaggle 6 - Renaming and Combining
-  rename() - lets you change index or column names
	- supports many input formats but most convenient is python dictionary
	- set_index used for index values
	- rename_axis used to change row and column index names
- 3 core methods for combining DataFrames and Series: concat, join, merge (most of what merge does can be done with join)
- concat - the simplest combining method, given a list of elements concat smushes them together along an axis
	- useful for different DataFrame or Series objects with same fields
- join - lets you combine different DataFrame objects with an index in common

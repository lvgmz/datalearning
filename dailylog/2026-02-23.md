### Kaggle 1 - Creating, Reading, and Writing
- pandas is a python library for data analysis
- 2 core objects
	- DataFrame - table containing array of individual entries, each with a certain value, each entry has a row and a column
	- Series  - sequence of data values, list, a single column of a DataFrame
- we often see data stored in files with csv being most common, csv - comma separated values - stores tables of values separated by commas
- you can assign var to a pandas read_csv() and use many commands on that csv variable like viewing how many records and columns, seeing first 5 rows
### Kaggle 2 - Indexing, Selecting & Assigning
- reviews.column_name shows all of a column of a DataFrame
- Series behave like dictionary and we use []\[] for column name + index to access a specific entry
- pandas have their own accessor operators loc and iloc
- iloc - follows index-based selection where selecting data is based on its numerical position
- loc - label-based selection selects data based on its index value not position so names based rather than number based
	- slicing behavior of loc and iloc is different, iloc is exclusive and loc is inclusive

- conditional selection - putting data object in a conditional statement returns list of true/false, this used in a loc returns only true data
- pandas has some built in selectors
	- isin - select data whose value is in a list of values
	- isnull - highlights values which are empty

- assigning data can be done by assigning a constant value or iterable values
### Kaggle 3 - Summary Functions and Maps
- there are many functions that summarize data in useful ways
- describe() - shows high level summary of attributes of given column
- mean() - average
- unique() - list of unique values
- value_counts() - list of all unique values and how often they occur

- maps - takes a set of values and maps them to another set of values done with:
	- map() - where we pass a lambda function and its done on all values, function should return single value
	- apply() - to transform whole DataFrame by calling a custom method on each row or column specified by axis value
- operators done on data objects like addition or subtraction are faster than maps but map functions are more advanced
